# NYC Restaurant Inspection Analysis (Tableau)

This Tableau project analyzes restaurant inspection data across New York City to identify safety trends, violation patterns, and cuisine performance.

## Objectives
- Identify boroughs with highest inspection issues
- Analyze cuisine risk levels
- Track inspection trends over time
- Evaluate violation severity

## Dashboards Included
- Inspection Overview
- Borough & Geographic Analysis
- Cuisine Type Performance
- Violation Deep Dive
- Safe to Serve Story

## Tools Used
- Tableau
- Data Visualization
- Data Cleaning
- Exploratory Analysis

## Key Insights
- Manhattan has highest inspection volume
- Pizza & BBQ cuisines show higher violation rates
- Grade B most common
- Critical violations ~35%
